# מחשבון גימטריא — APK אופליין

אריזת אפליקציית מחשבון הגימטריא כאפליקציית אנדרואיד **עצמאית לחלוטין** (Capacitor).
כל הקבצים — הקוד, מאגר התנ״ך (23,204 פסוקים), הפונטים — ארוזים בתוך ה-APK.
**עובדת בלי אינטרנט בכלל**, כולל על מכשיר שמעולם לא התחבר לרשת.

## איך משיגים את ה-APK (בלי Android Studio)

הבנייה מתבצעת אוטומטית בענן ב-GitHub Actions:

1. היכנס לריפו ב-GitHub → לשונית **Actions**.
2. בחר את ה-run האחרון של **"Build Android APK"** (ירוק = הצליח).
3. גלול למטה ל-**Artifacts** → הורד את **`gematria-apk`** (קובץ zip).
4. חלץ ממנו את `app-debug.apk`.
5. העבר את ה-APK למכשיר (כרטיס זיכרון / כבל) והתקן.
   - במכשיר: יש לאשר **"התקנה ממקורות לא ידועים"** (Settings → Security).

לבנייה חוזרת: Actions → "Build Android APK" → **Run workflow**.

## מבנה
```
www/                 קבצי האפליקציה (מועתקים מפרויקט gematria)
resources/           אייקון ומסך-פתיחה למקור
capacitor.config.json  הגדרות (appId, שם, צבעים)
.github/workflows/   בניית ה-APK בענן
```

## עדכון תוכן האפליקציה
עדכן את הקבצים ב-`www/` (מקור: פרויקט `gematria`), דחוף ל-GitHub, וה-APK ייבנה מחדש.
הבילד המקומי מסומן `<html class="median">` כדי להסתיר את הלוגו של לשקוד ואת באנר ההתקנה
(אין צורך בהם באפליקציה עצמאית).

## הערות
- ה-APK חתום במפתח debug — מתאים להתקנה ישירה (sideload). ל-Google Play צריך AAB חתום release.
- appId: `app.lishkod.gematria`
